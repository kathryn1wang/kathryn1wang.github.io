Project 4: IMAGE WARPING and MOSAICING

Kathryn Wang

Part 1: Image Rectification

Rectifying coaster to square

In my image warping and rectification approach, I manually define point correspondences between images and compute a homography matrix using least squares. This homography allows me to transform one image into the perspective of another. I construct a mapping matrix and using interpolation, I warp the entire image. I also compute the updated image size for the warping.

Part 3: Image Mosaic

For blending images into a mosaic, I use a simple weighted averaging technique to combine overlapping regions. After warping images to align within a larger canvas, each pixel is assigned a weight based on its visibility in the respective source images. The weights are calculated such that overlapping areas contribute equally to the final result. This basic approach results in a somewhat smooth mosaic.
